- Все элементы одного тензора должны иметь единый тип данных

```Python
import torch

# создаст трёхмерный тензор !но
t = torch.Tensor(3,5,2)
# !но в псоледних версиях pyTorch рекомендуется юзать empty
# формируется тензор 3х5х2 состоящий из вещественных 
# значений с типом float32 (32 бита/4байта)
t = torch.empty(3,5,2)
# если нужен иной тип данных указываем dtype
t = torch.empty(3, 5, 2, dtype=torch.int32)

# для создания тензоров рекомендуется юзать tensor()
t = torch.tensor([3,1])
# output: tensor(3,1)
t = torch.tensor([3, 2.0])
#output: tensor(3., 2.)
```
